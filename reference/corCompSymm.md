# Compound Symmetry Correlation Structure

Re-exports
[nlme::corCompSymm](https://rdrr.io/pkg/nlme/man/corCompSymm.html) from
the nlme package.

## Usage

``` r
corCompSymm(value, form, fixed)
```

## Arguments

- value:

  numeric vector of parameter values

- form:

  a one-sided formula of the form `~ 1 | g` specifying a grouping
  factor.

- fixed:

  unused
