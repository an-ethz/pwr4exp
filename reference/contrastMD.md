# Computes power of F-test for multi-dimensional contrast matrices

Computes power of F-test for multi-dimensional contrast matrices

## Usage

``` r
contrastMD(object, L, sig.level = 0.05, eps = sqrt(.Machine$double.eps))
```

## Arguments

- object:

  design object

- L:

  contrast matrix

- sig.level:

  significance level

- eps:

  numeric tolerance

## Value

A data frame
