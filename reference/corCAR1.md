# Continuous AR(1) Correlation Structure

Re-exports [nlme::corCAR1](https://rdrr.io/pkg/nlme/man/corCAR1.html)
from the nlme package.

## Usage

``` r
corCAR1(value, form, fixed)
```

## Arguments

- value:

  numeric value for the correlation parameter

- form:

  a one-sided formula of the form `~ t`, `~ 1 | g`, or `~ t | g`,
  specifying a time covariate `t`, a grouping factor `g`, or both.

- fixed:

  unused
